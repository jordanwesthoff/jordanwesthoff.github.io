Cloning into the Cluster.

Syncing the nodes to the cluster is as simple as cloning the thesis repo and then running a couple simple scripts:

$ cd /home/$user/Desktop && mkdir -p GitHub
$ cd /home/$user/Desktop/GitHub
$ git clone https://github.com/jordanwesthoff/senior_thesis.git

This repo has all of the cluster tools that have been developed so far. These include tools like Zabbix Monitoring and Slurm.
